#GiphyProject

This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).
*Because WebPack4 is not our friend yet*

### Install Dependencies
```sh
$ yarn install
```
- Axios
- Redux
- Thunk
- React-Router

### Start Project
```sh
$ yarn start
```
includes:
- SCSS build/watch
- ESLint
- StyleLint

### Production Build
```sh
$ yarn build
```
includes:
- Code Spitting
- Uglifying 
- Minifying
